# Personal Dotfiles

## Homebrew 

Install homebrew and then ansible then run 

```bash
make ansible-install
```

## Dotfiles

To install the configuration files. We can use: 

```bash 
make stow-install 
```

## Uninstall

```bash 
stow stow-uninstall 
```
